## 📊 Analisador de ROI de Campanhas de Marketing:
Este é um script Python simples e eficiente desenvolvido para calcular o Retorno sobre Investimento (ROI) de diferentes estratégias de marketing. O objetivo é fornecer uma visão clara de quais campanhas estão gerando lucro e quais estão resultando em prejuízo.

## 🚀 Funcionalidades:
Cálculo Automatizado: Processa uma lista de dicionários contendo dados de custo e receita.

Métrica de Desempenho: Utiliza a fórmula matemática do ROI para avaliar a eficiência financeira.

Visualização em Tabela: Formata os resultados em uma tabela limpa e legível no console.

🧮 A Lógica Matemática
O script utiliza a fórmula padrão de ROI para determinar o ganho obtido em relação ao valor investido:

$$ROI = \frac{\text{Receita} - \text{Custo}}{\text{Custo}}$$

### Nota: Um ROI de 1.00 significa que você dobrou seu investimento (100% de retorno), enquanto um valor negativo indica que a campanha não cobriu os próprios custos.
